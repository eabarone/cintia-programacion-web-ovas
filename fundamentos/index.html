Fundamentos de JavaScript

¿Te has preguntado cómo es que una página web puede mostrar mensajes, cambiar colores al hacer clic o validar lo que escribes en un formulario? Eso ocurre gracias a JavaScript, el lenguaje que da vida e interactividad a los sitios web. Si HTML organiza el contenido y CSS lo embellece, JavaScript se encarga de hacerlo dinámico y de responder a las acciones del usuario. Es como tener un robot, HTML te ayuda a construir su esqueleto, CSS le da la forma que tú decidas y JavaScript le da funcionalidades como hablar, saludar, etc.

En esta guía aprenderás los fundamentos esenciales para empezar a programar con JavaScript: cómo manejar variables, trabajar con distintos tipos de datos, usar operadores y expresiones y tomar decisiones con condicionales. Todo esto será la base para construir páginas web interactivas.

Objetivos de aprendizaje

  • 🧠 Comprender la función de JavaScript en el desarrollo web y su importancia para crear sitios interactivos y dinámicos.
  • 📝 Aplicar conceptos fundamentales de programación como variables, tipos de datos y operadores para manipular información en JavaScript.
  • 💻 Implementar estructuras condicionales para crear soluciones interactivas y automatizadas que respondan a diferentes situaciones.

A continuación encontrarás un tutorial interactivo completo sobre los fundamentos de JavaScript. Explora cada sección para aprender sobre variables, tipos de datos, operadores y estructuras condicionales.

💡 Consejo: Tómate tu tiempo para explorar cada sección del tutorial. Practica los ejemplos y experimenta con el código para reforzar tu aprendizaje.

Actividades propuestas

A continuación encontrarás tres ejercicios prácticos para aplicar lo aprendido sobre JavaScript. Para desarrollar las actividades debes usar la herramienta RunJs que se encuentra en el apartado de anexos. Esta herramienta permite encapsular todo el código en una sola función y luego ejecutarla.

🎮 Actividad 1: Juego de adivinanza

Un sitio web de entretenimiento quiere incluir un minijuego para enganchar a los usuarios. El sistema debe elegir un número secreto entre 1 y 10 y el jugador intentará adivinarlo escribiendo números en la pantalla. Cada vez que se equivoque, el sistema le dirá "intenta de nuevo" y solo cuando acierte mostrará el mensaje "¡Correcto, ganaste!", terminando el juego en ese momento.

💡 Pista: Usa un bucle while y la función prompt() para pedir números al usuario.

🔢 Actividad 2: Contador de pares

Una plataforma de apoyo escolar necesita un programa que muestre en consola todos los números pares entre 1 y 20, para que los estudiantes puedan identificarlos fácilmente sin que el profesor tenga que escribirlos uno por uno. El sistema debe recorrer los números en ese rango y mostrar únicamente los pares.

💡 Pista: Usa un bucle for y el operador módulo % para verificar si un número es par.

🎬 Actividad 3: Clasificación de edades

Un cine online quiere recomendar películas adecuadas según la edad de sus clientes. Para lograrlo, necesita un programa que pida la edad del usuario y lo clasifique en una categoría: "menor de edad" si es menor de 18 años, "adulto" si está entre 18 y 59 años, y "adulto mayor" si tiene 60 años o más, mostrando en pantalla un mensaje con el resultado.

💡 Pista: Usa estructuras condicionales if-else para clasificar la edad.

📌 Nota importante: Recuerda usar la herramienta RunJs disponible en el apartado de recursos/anexos para probar tu código.
