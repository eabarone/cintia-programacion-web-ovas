🌳 DOM en JavaScript

¿Alguna vez has entrado a una página web y al dar clic en un botón, cambia de color, aparece una animación o se abre una ventana emergente? Eso no ocurre por arte de magia ✨: detrás está el DOM (Document Object Model) y la manera en que los programadores lo manipulan para crear experiencias vivas en Internet.

El DOM es como un "árbol genealógico" 🌲 de la página web: cada título, párrafo, imagen o botón es una rama que podemos modificar en tiempo real. Cuando un usuario interactúa (por ejemplo, mueve el mouse, escribe en un formulario o presiona un botón), el programador puede decidir qué pasa en la página. ¡Así es como los sitios dejan de ser estáticos y se vuelven interactivos!

En esta unidad aprenderás cómo convertir páginas simples en experiencias dinámicas, dándole vida a tus ideas con JavaScript y el DOM 🚀.

Objetivos de aprendizaje

  • 🌳 Comprender la estructura del DOM como representación jerárquica de una página web y su importancia en la manipulación dinámica de contenido.
  • 🔧 Aplicar métodos de selección y manipulación del DOM para modificar dinámicamente el contenido, estilos y atributos de los elementos de una página web.
  • 🚀 Desarrollar aplicaciones web sencillas que resuelvan necesidades y situaciones reales del entorno digital mediante la manipulación del DOM.

A continuación encontrarás un tutorial interactivo completo sobre el DOM en JavaScript. Explora cada sección para aprender cómo seleccionar y manipular elementos de la página dinámicamente.

💡 Consejo: Tómate tu tiempo para explorar cada sección del tutorial. Practica los ejemplos y experimenta con la manipulación del DOM para reforzar tu aprendizaje.

Actividades propuestas

A continuación encontrarás tres ejercicios prácticos para aplicar lo aprendido sobre el DOM. Para desarrollar las actividades debes usar la herramienta Codi.link que se encuentra en el apartado de recursos. Esta herramienta integra HTML, CSS y JavaScript en un solo lugar.

🎨 Actividad 1: Personalizador de estados de ánimo

Una empresa de entretenimiento quiere personalizar su sitio web para que refleje diferentes estados de ánimo. En la página ya se encuentran tres botones con las emociones "Feliz", "Triste" y "Energético", y tu tarea consiste en acceder a esos elementos y al body de la página para modificar directamente el color de fondo, de modo que represente cada una de las emociones (por ejemplo amarillo para feliz, azul para triste, naranja para energético).

💡 Pista: Usa document.getElementById() para acceder a los botones y document.body.style.backgroundColor para cambiar el color de fondo.

🛒 Actividad 2: Carrito de compras virtual

En una tienda virtual se encuentra una lista de productos con un texto que dice "Agregar al carrito" al lado de cada uno, además de un contador en la parte superior que inicia en 0. Tu tarea es acceder a los elementos que representan los productos y al contador, simular que se agregaron tres productos actualizando el número en el contador, y modificar el texto de uno de los productos de la lista para que aparezca como "Oferta".

💡 Pista: Usa textContent o innerHTML para modificar el contenido de los elementos.

👋 Actividad 3: Formulario de bienvenida

Un colegio está creando una aplicación web para estudiantes y en el formulario ya existe un campo de texto para escribir el nombre y un espacio vacío donde se mostrará el mensaje de bienvenida. Tu tarea es acceder al campo del formulario, simular que un estudiante escribió "Ana" asignando ese valor desde el DOM, y mostrar en el espacio destinado el mensaje "¡Bienvenida, Ana!".

💡 Pista: Usa la propiedad value para asignar el valor al campo de texto y textContent para mostrar el mensaje.
